Module for Gravity Forms in Divi Builder.

== Description ==

Module for Gravity Forms in Divi Builder allows you to use and style Gravity Forms in the Divi Visual Builder.
Complete Gravity Forms style customization is finally possible, the Divi way!

Note: This plugin requires the Divi Theme (or Divi Builder plugin) and Gravity Forms.

* [Divi Theme](https://www.elegantthemes.com/affiliates/idevaffiliate.php?id=33948&url=58899&tid1=wordpress_org)
* [Gravity Forms](https://rocketgenius.pxf.io/c/2060242/445235/7938)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/module-for-gravity-forms-in-divi-builder` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. You're Done! You will notice that a new "Gravity Form" module is present in the Divi Builder.
